# Ejecutar en desarrollo
1. clonar el repositorio
2. Ejecurar
 yarn install

3. tener nest CLI instalado

npm i -g @nestjs/cli


4. levantar la base de datos

docker-compose up -d

5. clonar el archivo __.env.template__

6.reconstruir la base de datos con la semilla
